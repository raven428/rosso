Rosso
=============
![porco][irj]

Rosso is a C utility that sorts FAT32 file systems. Many hardware players do not
sort files automatically but play them in the order they were transferred to the
device.

Supported platform
------------------
Windows 64-bit

Requirements
------------
[StdLib][csh]
